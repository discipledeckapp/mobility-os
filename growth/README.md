# Mobiris Growth Engine

This folder is a practical, low-cost growth operating system for Mobiris Fleet OS.

It is designed for founder-led or very small-team execution using:

- Codex for structuring, enrichment, writing, and workflow logic
- Google Sheets as the lead database / lightweight CRM
- Gmail for outreach support
- LinkedIn and X for manual or semi-manual lead discovery
- WordPress for blog publishing, landing pages, campaign pages, and lead capture

This system is intentionally built around:

- operator pain, not generic SaaS messaging
- legal and platform-safe workflows
- low-volume, high-quality personalization
- measurable handoffs from lead -> reply -> demo -> activation
