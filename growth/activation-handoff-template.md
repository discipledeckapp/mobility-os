# Activation Handoff Template

Use this when a lead moves from positive reply or demo into signup and early onboarding.

## 2. Expected First Win

Pick one:

- first driver added
- first driver verified
- first assignment created
- first remittance workflow reviewed

## 3. Handoff Note

```text
This lead came in because they care most about [PAIN].

The message that resonated was [ANGLE].

In the first onboarding touch, show them how Mobiris helps them get to [FIRST WIN] as quickly as possible.
```

## 4. Activation Questions To Ask

- how are you currently tracking drivers?
- how do you currently know who had which vehicle?
- how do you currently track remittance or repayment?
- where do disputes happen most often?
- what would a useful first win look like in 7 days?

## 5. Feedback To Send Back To Growth

- what promise brought them in?
- what confused them after signup?
- what language did they use to describe the problem?
- what blocked first assignment creation?
